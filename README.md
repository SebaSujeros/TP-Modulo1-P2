# Laberinto - TP Módulo 1 P2

## Descripción
Juego de aventura top-down desarrollado en Phaser 4. 
Un explorador debe recorrer una serie de laberintos recolectando llaves 
para poder avanzar al siguiente nivel.

## Objetivo
Recolectá las llaves distribuidas por el laberinto y llegá a la salida. 
Necesitás juntar todas las llaves del nivel para poder pasar al siguiente.

## Niveles
- **Nivel 1:** Encontrá 5 llaves y llegá a la salida.
- **Nivel 2:** Llevás las llaves del nivel anterior. Encontrá 5 llaves más (total 10) y llegá a la salida.
- **Nivel 3:** Mapa de mayor tamaño con movimiento de cámara. Encontrá 5 llaves más (total 15) y llegá a la salida.

## Controles
| Tecla | Acción |
|-------|--------|
|   ←   | Mover izquierda |
|   →   |  Mover derecha  |
|   ↑   |  Mover arriba   |
|   ↓   |  Mover abajo    |

## Tecnologías
- [Phaser 4](https://phaser.io/)
- [Tiled](https://www.mapeditor.org/) para la creación de mapas
- [TexturePacker](https://www.codeandweb.com/texturepacker) para el spritesheet
